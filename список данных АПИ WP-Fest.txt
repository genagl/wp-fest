Данные, которые аккумулирует и передаёт по API комплекс WP-Fest:
1. Данные Фестиваля:
	- название
	- приветственный текст (описание)
	- УРЛ-адрес
	- статус (до начала - 0, период разработки проектов - 1, период оценки проектов экспертами - 2, фестиваль закрыт - 3)
	- настройки 
	- доступные роли участников
	- список оценочных критериев
	- список жанров, по которым оцениваются Проекты
2. Проеты;
	- название
	- дата создания
	- командпа (автор проекта, наставник, участники)
	- ханры Проекта
	- связанные ссылки в соц. сетях
	- оценка каждого из Экспертов по каждому критерию
	- средняя оценка по каждому критерию
	- средняя оценка Проекта
	- журнал Проекта (каждый пост - заголовок, текст, дата, автор сообщения, доступность (только для команды или публичный)
3. Участники (Пользователи);
	- публичное имя
	- внешняя соц. сеть, через которую была создана учётная запись
	- набор ролей из тех, которые включены в данном Фестивале (Администратор, Эксперт-оценщик, Автор проектов, Уастник команд, Наблюдатель)
4. Жанры
5. Критерии
	- обобщающая категория, которой приписан критерий
6. Обобщающие оценочные категории
7. Эксперты
	- оценка, поставленная каждому Проету по каждому Критерию
	- комментарий к оценке (Почему Эксперт поставил именно такую)
	- общее напутствие Эусперта создателям кажого Проекта
